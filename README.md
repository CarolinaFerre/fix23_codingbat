# fix23_codingbat
ejercicio de arrays_fix23_arrays1_codingbat


Dada una longitud de matriz int 3, si hay un 2 en la matriz seguido inmediatamente por un 3, establezca el elemento 3 en 0. Devuelva la matriz modificada.
fix23([1, 2, 3]) → [1, 2, 0]
fix23([2, 3, 5]) → [2, 0, 5]
fix23([1, 2, 1]) → [1, 2, 1]
